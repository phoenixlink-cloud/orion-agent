## Documentation Conventions

Throughout this documentation:

- `monospace` indicates code, commands, or file paths
- **Bold** indicates important terms or UI elements
- *Italic* indicates emphasis or new terms
- > Blockquotes indicate tips or important notes

## Contributing to Documentation

Found an error? Want to improve something?

1. Fork the repository
2. Edit the relevant `.md` file in `docs/`
3. Submit a pull request

Documentation contributions still require a signed [CLA](../CLA.md), but the process is simpler for docs-only changes.
